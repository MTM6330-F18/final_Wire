Responsive Web Design

The ubiquity of devices of all shapes and sizes means web content can no longer be guaranteed to be delivered on a specific platform. Students are introduced to responsive web design, using HTML, CSS and modern web standards to build sites that work on any size device. Students apply current best practices in web design and coding techniques, including accessibility and search engine optimization.


Code: MTM6130
